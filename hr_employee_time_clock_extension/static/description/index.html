Employee Time Clock Extension provides opportunity to use overtime settings for each employee and allows to configure overtime settings.

Once you install the module, you'll see additional checkbox on the employee contract working schedule page called Use Overtime Setting. On checking this button you can observe additional tab called Overtime settings which allows you to set up:

  • "Percent Count" which represent salary increment;
  • "Minimum overtime days" which allows to set up how many days employee should work to have an increased salary;
  • Days of week and overtime working hours, start/end date for each weekday;
  • Shift between two days which allows to use night shift between two days.

Features

  • Calculates "bonus hours", "night shift hours" based on individual employee contracts
  • Additional fields in attendance sheet
  • Bonus hours are included in attendance sheet field named "Total Balance"

How it works

The module adds a new data types called bonus hours and night shift hours, which is the baseline for all calcualtions. Bonus hours and night shift hours are calculated based on the overtime schedule assigned to an employee/contract. Whenever an employee works on overtime hours, bonus hours and night shift hours for those days are set to configured count.

Once you have a overtime schedule set up for an employee, his bonus hours, night shift hours are tracked using sign in/out timestamps. Each employee can use one of these methods to sign in/out:

  • Sign in/out to your Odoo instance
  • Sign in/out from our iPhone app
  • Sign in/out at the company's entrance using our QR code scanning iPhone/iPad app

How to configure overtime settings

To allow employees use overtime settings in system you'd need to:

  • create contract for employees;
  • set up contract working schedule.

Set up contract working schedule

Follow below steps to set up contract working schedule.

  • Open Odoo's Employees page and switch to the Contracts tab.
  • Select contract and go to Other information tab -> press on Working Schedule value.
  • Check Use Overtime Setting button.
  • Go to Overtime settings tab and set up overtime configurations.
  • Press Initial Employee Overtime button to calculate employees' overtime.
  • Press Save button to save overtime configurations.
